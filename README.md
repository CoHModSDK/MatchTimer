# Match Timer

## 📜 Description

Match Timer is a mod for `Company of Heroes Relaunch` built on top of `CoHModSDK` which displays the match timer in-game.

## 🔧 Installation

1. Install the `CoHModSDK` loader in your game directory
2. Copy `MatchTimer.dll` into the `mods` folder
3. Add `MatchTimer.dll` to `CoHModSDKLoader.ini`
5. Start the game
